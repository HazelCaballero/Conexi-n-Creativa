# Conexión Creativa

Conexión Creativa es una plataforma digital diseñada para fomentar el intercambio de conocimiento, bienes y servicios 
entre mujeres mediante un sistema de trueque. Este proyecto busca crear una comunidad inclusiva y colaborativa donde 
las usuarias puedan aprender, crecer y apoyarse mutuamente.

## Características del Proyecto

- **Intercambio de Trueque:** Permite a las usuarias intercambiar bienes y servicios sin necesidad de dinero, aunque 
no limita su uso si así se desea.
- **Comunidad Inclusiva:** Diseñado para mujeres de todas partes, promoviendo la colaboración y el apoyo mutuo.
- **Información del Proyecto:** Incluye detalles como ubicación, horario de atención y contacto.

## Estructura del Proyecto

### Componentes

1. **HomeByHazelCC.jsx**
   - Contiene la estructura principal de la página de inicio.
   - Muestra información sobre el proyecto, como descripción, ubicación, horario, y contacto.

2. **FoundressByHazelCC.jsx**
   - Componente que muestra información sobre la fundadora del proyecto.
   - Incluye una imagen y un botón para redirigir a la página de contacto.

3. **HeaderByHazelCC.jsx**
   - Componente para la cabecera del sitio web.
   - Incluye el logo y la navegación principal.

4. **FooterByHazelCC.jsx**
   - Componente para el pie de página.
   - Contiene enlaces de contacto y redes sociales.

5. **AsideByHazelCC.jsx**
   - Barra lateral que organiza enlaces o información adicional.

6. **LoginByHazelCC.jsx**
   - Formulario para que las usuarias inicien sesión en la plataforma.

7. **SignByHazelCC.jsx**
   - Formulario para que las usuarias se registren en la plataforma.

8. **ContactByHazelCC.jsx**
   - Componente que contiene un formulario de contacto para enviar mensajes o consultas.

9. **BarterByHazelCC.jsx**
   - Componente que muestra el contenido del muro o funcionalidades de intercambio.

10. **AgregarProducto.jsx**
    - Componente para agregar nuevos productos o servicios al sistema.

11. **ProfileByHazelCC.jsx**
    - Componente que muestra y permite gestionar la información del perfil de la usuaria.

12. **SeccionDestacada.jsx**
    - Componente que resalta secciones importantes o destacadas del sitio.

### Páginas

1. **Home.jsx**
   - Página principal que incluye el componente `HomeByHazelCC` y otras secciones destacadas.

2. **Fundadora.jsx**
   - Página dedicada a la fundadora, que incluye el componente `FoundressByHazelCC`.

3. **Comunicate.jsx**
   - Página de contacto que incluye el formulario de contacto y otros componentes relacionados.

4. **Inicio.jsx**
   - Página de inicio de sesión y registro, que incluye los componentes `LoginByHazelCC` y `SignByHazelCC`.

5. **Muro.jsx**
   - Página que organiza el contenido del muro, incluyendo el componente `BarterByHazelCC`.

6. **Perfil.jsx**
   - Página del perfil de la usuaria, que incluye el componente `ProfileByHazelCC`.

7. **AdminTools.jsx**
   - Página de herramientas de administración, que incluye el componente `AgregarProducto`.

## Tecnologías Utilizadas

- **React:** Biblioteca de JavaScript para construir interfaces de usuario.
- **React Router:** Para manejar la navegación entre las páginas de la aplicación.
- **SweetAlert2 (swal):** Para mostrar alertas personalizadas y modales interactivos.
- **React Hooks:** Hooks como `useState`, `useEffect`, y `useNavigate` para manejar el estado y la lógica de navegación.
- **Vite:** Herramienta de desarrollo rápida para proyectos de frontend.
- **CSS:** Para el diseño y la personalización visual.

## Cómo Ejecutar el Proyecto

1. **Clonar el Repositorio:**
   ```bash
   git clone https://github.com/HazelCaballero/Conexi-n-Creativa.git
   cd CreativeConnetion
   ```

2. **Instalar Dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el Proyecto:**
   ```bash
   npm run dev
   ```

4. **Abrir en el Navegador:**
   - Accede a `http://localhost:3000` (o el puerto indicado en la consola).
